Каждое поле бланка МК должно иметь название. Для удобства взаимодействия названия переменных, обозначающих эти поля должны быть одинаковыми для всех уровней разработки. Этот файл должен содержать все имеющиеся переменные этих полей. Предлагается использовать смешанный шрифт с префиксом, обозначающим, что данная переменная хранит содержимое поля МК.

mk_Complaints    - жалобы
mk_Anamnez       - анамнез
mk_StatusLocalis - Status Localis
